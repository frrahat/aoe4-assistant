Instructions:
  • Upload Image: Select a screenshot image using the file input above
  • Set Scaling: Enter target width and height for coordinate scaling and click "Apply Scaling"
  • Grid Controls: Adjust grid size and opacity using sliders, or toggle grid visibility
  • View Mode: Hover over the image to see both actual and scaled coordinates in a transparent popup
  • Crop Mode: Click "Enable Crop Mode" to switch to rectangle selection mode
  • Draw Selection: In crop mode, click and drag to draw a selection rectangle on the image
  • Crop Dialog: After drawing, a dialog shows coordinates and dimensions with auto-generated filename
  • Download: Confirm to download the cropped area as PNG, or cancel to discard selection
  • Filename Format: Default naming: YYYY-MM-DD_HH-MM-SS_x_y_width_height (all coordinates are scaled)
